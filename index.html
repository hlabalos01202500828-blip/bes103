LESSON 1 — COMPUTER ORGANIZATION

🔩 4 Elements of Computer System

  • Hardware — Physical components
  • Software — Programs & OS
  • Data — Raw input/output
  • Communications — Network links

🧠 CPU Architecture

  • Control Unit (CU) — Fetch/Decode/Execute
  • ALU — Arithmetic & Logic operations
  • Registers — Fastest on-chip storage
  • Bus Interface Unit — Data movement

💾 Storage Hierarchy

  • Registers → Cache (L1/L2/L3)
  • RAM (Primary) → Virtual Memory
  • Secondary (HDD/SSD/Optical)
  • Faster = Smaller & More Expensive

💻 Computer Categories

  • Desktop, Workstation, Mobile/Tablet
  • Smartphone, Game Console
  • Network Server, Mainframe
  • Supercomputer (El Capitan #1)

⚡ El Capitan Supercomputer

  • 2.746 exaFLOPS peak performance
  • Lawrence Livermore NL, California
  • 11M+ AMD MI300A processors
  • Cost: ~$600M | Power: 29.5 MW

🔄 System Flow

  • CPU ↔ Bus Interface Unit
  • I/O Bridge connects subsystems
  • Memory Bus → RAM
  • I/O Bus → Peripherals
LESSON 4 — INTERNET BASICS

🌐 Internet History

  • 1958: DARPA founded (post-Sputnik)
  • 1969: ARPAnet — 4 nodes
  • 1991: WWW released by Tim Berners-Lee
  • 2019: 4.39B internet users worldwide

🔌 LAN Components

  • MAC Address — Physical hardware ID
  • IP Address — Logical network ID
  • DHCP — Auto IP assignment
  • Switch, Modem, Gateway, Router

🕸️ WWW Structure

  • Structural: Client/Server/Cache/Internet
  • Semantic: HTTP/HTML/XML/URI
  • Client — Browser (requests pages)
  • Server — Responds with content

🔗 URL Anatomy

  • Protocol (http/https)
  • Domain Name (e.g., google.com)
  • Path (/folder/page.html)
  • Query String (?key=value)

☁️ Cloud vs On-Premise

  • Cloud: SaaS, PaaS, IaaS
  • Pay-per-use vs Capital Expense
  • Scalable vs Fixed Capacity
  • Provider-managed vs Self-managed

🛡️ Security & Protocols

  • Firewall — Network traffic filter
  • Cookies — Session state tracking
  • FTP — File Transfer Protocol
  • DNS — Domain → IP resolution
